# Simple Django Project

This project includes:

- User registration
- User login
- Home page
- Dashboard (login required)

## Quick Start

1. Install dependencies:
   ```
   pip install django
   ```

```
2. Run migrations:
```

python manage.py migrate

```
3. Start the server:
```

python manage.py runserver

```
4. Visit http://127.0.0.1:8000/
```
